- **PHIND AI GPT**

This bang allows one to ask a question to the gpt based AI tool called [phind](https://www.phind.com/) via duckduckgo search bar.
```
Site name : Phind AI
Bang command : phind
Bang URL : https://www.phind.com/search?q={{{s}}}
Bang category : Online Services > Tools
```
